# Setting up reverse proxy using Nginx

Nginx is a high-performance web server and reverse proxy server that can be used to handle incoming requests and route them to the appropriate backend services. In this section, we will set up Nginx as a reverse proxy for our application.

This will allow us to expose our application to the internet and handle incoming requests efficiently.


- Log in to the VM.
- Run the setup script [reverseproxy.sh](../infra/shell_scripts/reverseproxy.sh) as sudo/superuser using ` sudo chmod +x ./reverseproxy.sh &&sudo ./reverseproxy.sh` to configure Nginx as a reverse proxy for your application.
- The script will install Nginx, set up the necessary reverse proxy rules to forward requests to your application.
- The script will also create a self-signed SSL certificate for HTTPS support.
- You can add more rules to the Nginx configuration file as needed.
- You can also access your application using the domain name if you have set up a DNS record pointing to your VM's public IP address.
