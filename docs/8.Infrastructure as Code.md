# Infrastructure as Code (IaC) with OpenTofu

Tired of managing your infrastructure manually? Want to automate the process and make it more efficient and spin the infra for different environment for DEV, STAGING, PROD etc with one command ?  Look no further than OpenTofu!

OpenTofu is an open-source alternative to Terraform, a popular Infrastructure as Code (IaC) tool. It allows you to define and manage your infrastructure using code, making it easier to automate and version control your infrastructure changes.


## To get started with OpenTofu, follow these steps:

### Confifure your environment
- Install OpenTofu using `brew install opentofu` or refer installation [instructions](https://opentofu.org/docs/intro/install/) for your OS.
- Create a directory such as [opentofu](../infra/opentofu/)
- With the help of [terraform.tfvars.example](infra/opentofu/terraform.tfvars.example) create a terraform.tfvars file with your own values. This file contains the variables that will be used in your OpenTofu configuration.
- Modify the modules as per your needs. The modules are defined in the [main.tf](../infra/opentofu/main.tf) file. This file contains the main configuration for your OpenTofu project, including the resources you want to create and their dependencies.

### Initialize and run OpenTofu
- Initialize the OpenTofu project using `tofu init`. This command will download the necessary provider plugins and set up the backend for your project.
- Validate the configuration using `tofu validate`. This command will check your configuration for any errors or issues.
- Plan the changes using `tofu plan`. This command will show you what changes will be made to your infrastructure based on your configuration.
- Apply the changes using `tofu apply`. This command will create or update the resources defined in your configuration.
- Destroy the resources using `tofu destroy`. This command will remove all the resources created by your OpenTofu project.
- For more information on how to use OpenTofu, refer to the [documentation](https://newtuple.atlassian.net/wiki/spaces/DAKB/pages/308281345/Terraform+OpenTofu).
